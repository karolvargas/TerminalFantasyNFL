## To run program:

- First:
```
 cd nflgame2
```
then run
```
 python2 fantasyFootballPlay.py
```
  Enter whether you want to see Offense or Defense player stats

 
- Second:
  Enter what stats you want to see
 
- Third:
  Enter the year and week of the NFL season(up to 2017) as the following format
  ex. 2016 5

- Fourth:
  Enjoy the results
